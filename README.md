# esxtop Viewer

Visualize esxtop data captured in batch mode.

## Requirements

- Node.js 22+
- Corepack

## Build

```sh
corepack enable
corepack yarn install
corepack yarn run check
corepack yarn build
```

## Development

```sh
corepack enable
corepack yarn install
corepack yarn dev
```

## Toolchain

This project uses Vite+ via the local `vp` CLI exposed through Yarn scripts.
